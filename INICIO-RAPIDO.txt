❓ PROBLEMAS?
─────────────────────────────────────────────────────────────────

Porta em uso:
  lsof -i :5173  (Frontend)
  lsof -i :8080  (Backend)
  lsof -i :5432  (Database)
  kill -9 <PID>

Database não existe:
  docker exec taskscheduler-postgres psql -U postgres \
    -c "CREATE DATABASE taskscheduler;"

Build falhou:
  cd backend && mvn clean install -DskipTests

Dependencies corrompidos:
  cd frontend && rm -rf node_modules && npm install
